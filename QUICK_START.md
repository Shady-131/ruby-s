# 🚀 Quick Start (1-2-3 Steps)

## Step 1️⃣: Install Dependencies
```bash
npm install
```

## Step 2️⃣: Start Development Server
```bash
npm run dev
```

## Step 3️⃣: Open in Browser
Visit: **http://localhost:3000**

---
